# Final Project - 離線多人棋盤對戰平台

👉 **版本**: 0.0.3
👉 **系統需求**: iOS 17.0+
👉 **開發框架**: Swift / SwiftUI

> ⚠️ **開發狀態提醒**
> 本專案目前處於早期開發階段（v0.0.3），核心架構已建立，但仍有許多 Bug 尚未修復。此版本目前僅供**簡單展示 (Demo)** 與概念驗證使用。

這是一個以**高擴充性**為目標的 iOS 原生雙人對戰平台。玩家可在無網際網路的狀況下，透過藍牙或點對點 Wi-Fi 尋找附近玩家，建立房間並進行多款棋盤遊戲。

## 🌟 核心特色

- **離線連線對戰**：基於 Apple 的 `MultipeerConnectivity`，無需伺服器即可近距離連線。
- **模組化遊戲架構**：獨創 `GameEngine` 協議，未來加入象棋、西洋棋等新遊戲只需實作單一協議，不需要重寫大廳或房間邏輯。
- **房主規則同步**：建立房間後，由「房主」統籌選擇遊玩的遊戲與詳細規則（如棋盤大小、禁手等），設定完成後自動同步給對手。
- **全局聊天系統**：遊戲中右下角的浮動聊天按鈕，對方傳訊時會彈出 4 秒快顯通知，不干擾遊戲進行。支援自訂快捷回覆與自由文字輸入。
- **防誤觸的落子系統**：所有遊戲皆支援「點擊預覽 → 右下角確認」的兩步操作機制，配合固定高度的底列，確保下棋時版面穩定。
- **開發者測試模式**：首頁提供「測試模式」，可在單機上模擬雙方連線與聊天，方便後續快速開發與抓蟲。

## 🎮 目前支援遊戲

### 1. 黑白棋 (Reversi)
- 自動標示可落子位置、自動翻轉對手棋子。
- 無步可走時自動跳過回合。
- 支援自由設定棋盤大小（6×6、8×8、10×10、12×12）。

### 2. 五子棋 (Gomoku)
- 真實還原的星位與交叉線棋盤。
- 支援最高 25×25 超大棋盤。
- 大棋盤支援手勢雙指縮放、拖拉平移，並支援長按精準落子。
- 提供自訂「禁手規則（三三、四四、長連）」開關。

## ⚙️ 檔案架構概覽
- `Core/`：包含無關遊戲種類的核心機制，如 `LobbyView` (大廳)、`RoomView` (房間)、`MultipeerManager` (網路連線)、`ChatManager` (聊天)。
- `Games/`：各個遊戲的獨立實作包，每個遊戲都擁有專屬的 Model、Engine 與 View。
