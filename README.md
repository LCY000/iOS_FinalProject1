# Final Project - 離線多人棋盤對戰平台

👉 **版本**: 0.1.0
👉 **系統需求**: iOS 26.2+
👉 **開發框架**: Swift / SwiftUI

這是一個以**高擴充性**為目標的 iOS 原生雙人對戰平台。玩家可在無網際網路的狀況下，透過 Wi-Fi 直連尋找附近玩家，建立房間並進行多款棋盤遊戲。

## 🌟 核心特色

- **離線連線對戰**：基於 Apple 的 `MultipeerConnectivity`，無需伺服器即可近距離連線。
- **模組化遊戲架構**：獨創 `GameEngine` 協議，未來加入象棋、西洋棋等新遊戲只需實作單一協議，不需要重寫大廳或房間邏輯。
- **房主規則同步**：建立房間後，由「房主」統籌選擇遊玩的遊戲與詳細規則（如棋盤大小、禁手等），設定完成後自動同步給對手。
- **再來一場投票**：遊戲結束後雙方可發起投票重賽，對方接受則直接重開、拒絕則回房間，不再卡住等待。
- **全局聊天系統**：遊戲中右下角的浮動聊天按鈕，對方傳訊時會彈出 4 秒快顯通知，不干擾遊戲進行。支援自訂快捷回覆與自由文字輸入。
- **防誤觸的落子系統**：所有遊戲皆支援「點擊預覽 → 右下角確認」的兩步操作機制，配合固定高度的底列，確保下棋時版面穩定。
- **開發者測試模式**：首頁提供「測試模式」，可在單機上模擬雙方連線與聊天，方便快速開發與除錯。

## 🎮 目前支援遊戲

### 1. 黑白棋 (Reversi)
- 自動標示可落子位置、自動翻轉對手棋子，並以紅色環標示上一步落點。
- 無步可走時自動跳過回合。
- 支援自由設定棋盤大小（6×6、8×8、10×10、12×12）。

### 2. 五子棋 (Gomoku)
- 真實還原的星位與交叉線棋盤，以標記點顯示上一步落點。
- 支援最高 25×25 超大棋盤，雙指縮放與拖拉平移。
- 提供可自訂的禁手規則：**三三**、**四四**（僅計活四）、**長連**，可分別指定適用對象（僅黑方／雙方）。

## ⚙️ 檔案架構概覽

```
    └── Gomoku/    # 五子棋 Model + Engine + View
```
